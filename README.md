# Proyecto Lazarus

Aplicación de escritorio desarrollada con Lazarus / Free Pascal.

## Requisitos

- Lazarus
- Free Pascal Compiler

## Cómo abrir el proyecto

1. Abrir Lazarus.
2. Ir a `Project > Open Project`.
3. Seleccionar `project1.lpi`.
4. Compilar y ejecutar.

## Archivos principales

- `project1.lpr`: archivo principal del programa.
- `project1.lpi`: archivo del proyecto Lazarus.
- `unita.pas`: código del formulario.
- `unita.lfm`: diseño visual del formulario.
